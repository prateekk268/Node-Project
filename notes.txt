-> what is a graphQL?
GraphQL is a query language for APIs
 It can be used with:
 . with you existing data 
 . external APIs built upon RESTful principles.

 It is NOT a query to your database

-> Comparison RESTful 
RESTful                                       GraphQL
GET,POST,PUT,PATCH,DELETE                     Query or Mutation

GraphQL has following methods
Query - Read operations
Mutation - Write operations
Subscriptions - Realtime communications
